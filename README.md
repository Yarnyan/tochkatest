Клонирование и запуск:

1. Откройте повершел и впишите git clone https://github.com/Yarnyan/tochkatest.git
2. Откройте проект в папке, напишите npm i, следом npm run dev.
3. У вас откроется игра

Функционал игры:

Что бы играть, надо нажимать на стрелочки в самом верху окна игры!

Игровой процесс:

1. Игра представляет собой классическую игру 4 в ряд, где два игрока по очереди делают ходы.

2. Для этого нужно нажимать на стрелочки в верхней части игрового поля. Каждое нажатие активирует соответствующую ячейку в колонке.

3. Цель игры — собрать 4 фишки подряд по вертикали, горизонтали или диагонали.

Игра против ИИ:

1. В игре реализован базовый ии, с которым можно сыграть.

2. ИИ выполняет ходы через определённые интервалы времени.

Анимации:

1. Анимация победы или ничьи, которая отображается при завершении игры.

Функционал отмены хода:

1. Реализована возможность отмены последнего хода (Undo). При нажатии на кнопку отмены, предыдущий ход будет отменён, и игроку предоставится возможность сделать новый ход.

Определение победителя:

Игра автоматически определяет победителя, если один из игроков собрал 4 фишки подряд.

В случае ничьей, когда все ячейки заняты, но никто не выиграл, игра уведомит об этом.

Технический стек проекта

React с TypeScript: Используется для создания компонента игры, обработки логики игры и состояний.

Tailwind CSS: Для удобной и быстрой стилизации компонентов интерфейса.

Lottie: Для создания анимаций, таких как падение фишек и анимация победы.
